Admin Tools

Control Your Operation

A purpose-built admin dashboard with the controls range owners actually need — not a repurposed generic CRM.

Admin Dashboard

At-a-glance view of today's check-ins, upcoming events, recent applications, and key metrics. Everything an RO or owner needs to see when they open the app.

Staff Management

Add RSOs, match directors, and administrative staff. Assign roles with clear responsibilities — staff members appear with their role badges throughout the system.

Granular Permissions (RBAC)

Control exactly what each staff member can access. Toggle permissions per section — Events, Members, Scoring, Commerce, Settings, and more. Only the range owner manages permissions.

Range Configuration

Set up your range identity — name, location, bay/lane configuration, branding. Configure the operational parameters that define how your range runs.

Bulk Import

Migrating from spreadsheets? Import your existing member list via CSV. Map columns to member fields, validate data, and bring your records into Range Command in one step.

ID Card Generation

Generate member ID cards in bulk with member photos, numbers, and range branding. Export for printing or digital distribution.

Members

Membership That Runs Itself

From the first application to daily check-ins — a complete member lifecycle managed in one system.

Membership Applications

New members apply directly in the app. Applications include all required information, go through an approval workflow, and automatically create member profiles on acceptance.

Member Verification

Verify members by phone number, member number, or name. Handles format variations — stripped phone digits, member number prefixes, partial matching. Designed for real-world check-in scenarios.

Self Check-In

Members check themselves in on their own phone. Select a location or bay, tap Check In, done. Shows up instantly on the admin's live check-in feed.

Admin Check-In

Member doesn't have their phone? No problem. Admins can search by name or member number and check them in from the live check-in screen. Same record, same tracking.

Member Profiles

Complete member records — contact information, membership status, roles, check-in history, event participation, and custom notes. Everything you need about a member in one place.

Member Directory

Searchable directory with filters for status, roles, and membership type. Find any member instantly. View counts, export data, manage in bulk.

Events

Competitions Without the Chaos

Create events, manage registration, run matches, and finalize results — without a stack of clipboards and a spreadsheet back at the office.

Event Creation

Set up competitions with event type, date, location, registration deadlines, capacity limits, and fee structure. Publish to your members instantly.

Online Registration

Members register through the app. Payment collected at registration via Stripe. Waitlist support when events fill up. Automatic confirmation notifications.

Match Setup

Configure stages, scoring parameters, divisions, and classes. Set up the structure before match day so you can focus on running the event, not building spreadsheets.

Event Calendar

Members see all upcoming events, registration status, and their registered events in one view. No more "when is the next match?" questions — it's always visible.

Scoring

Real-Time Scoring. Instant Results.

Enter scores as stages complete. Results update live. Participants get push notifications when the match is finalized.

Stage Scoring

Score each stage as it completes. Multiple scorers can work simultaneously across different stages. Data syncs in real-time across all devices.

Division & Class Support

Score across multiple divisions and classes within the same match. Separate rankings calculated automatically — no manual sorting required.

Match Finalization

Review all scores, resolve disputes, and finalize results with one action. Finalization locks scores, updates leaderboards, and triggers push notifications to all participants.

Push Notifications

When a match is finalized, every participant receives a push notification with their results. No more checking the bulletin board — results come to them.

Score History

Complete scoring history for every member across all events. Track performance over time, identify trends, and maintain official records.

Dispute Resolution

Score corrections with audit trail. If a score needs adjustment after entry, the change is logged with who made it and why — full accountability.

Leaderboards

Rankings That Update Automatically

No more manually tallying season standings. Leaderboards calculate from match results in real time.

Season Tracking

Define seasons with date ranges and scoring rules. Leaderboards accumulate results across all matches within a season. Members watch their ranking change after every event.

Discipline Filtering

Filter leaderboards by discipline, division, or class. Members see where they stand in their specific category — not buried in a list of unrelated divisions.

Member Rankings

Each member sees their personal ranking, season history, and performance progression. Competition drives engagement — visible rankings keep members coming back.

Public Visibility

Leaderboards visible to all range members. Promotes healthy competition and gives members a reason to participate in more events.

Reporting

Transaction History

Operational Dashboards

Real-time dashboard views showing today's check-in count, upcoming events, pending applications, and key operational metrics. Open the app and know the state of your range.
